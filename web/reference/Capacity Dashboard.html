2 · Per-engineer baseline

Where the week goes

The slice of a normal week that never reaches project work — meetings, code review, time off. We count each overhead once. Drag toward your team's ideal to see capacity free up.

3 · Team reservations

KTLO & recurring work

KTLO“Keep The Lights On” — recurring upkeep that happens no matter what projects are running. is work reserved before any project: support tickets, incidents, interviews, onboarding others. Whatever's left is real project capacity.
